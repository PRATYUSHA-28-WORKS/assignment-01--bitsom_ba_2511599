## ETL Decisions

### Decision 1 — Date Standardization
Problem: The dataset had inconsistent date formats.
Resolution: Converted all dates into a standard YYYY-MM-DD format.

### Decision 2 — Handling NULL Values
Problem: Some fields had missing values.
Resolution: Replaced NULL values with default values or removed incomplete records.

### Decision 3 — Category Normalization
Problem: Product categories had inconsistent casing (e.g., electronics, Electronics).
Resolution: Standardized all categories to proper case for consistency.
